# Country Place Community Website

Static website for Country Place MHP, LLC — a mobile home, RV, and cabin community in Hico, TX.

**Live site:** [countryplacecommunity.com](https://www.countryplacecommunity.com)

## Structure
```
index.html    — Home page
about.html    — About the community
cabins.html   — Cabins & sites for rent
contact.html  — Contact form & info
css/style.css — All styles
js/main.js    — Nav toggle & form handling
```

## Hosting
Hosted on Cloudflare Pages, deployed from this repo.

## Contact Form
The contact form uses [Formspree](https://formspree.io). Update the form action URL in `contact.html` with your Formspree endpoint.

## Making Changes
Edit the HTML/CSS files and push to `main`. Cloudflare Pages auto-deploys.
